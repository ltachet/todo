# Todo Visualizer (Next.js)

A minimal Next.js + React todo app that lets you:
- Add tasks
- Mark tasks complete
- Delete tasks
- Filter by all/active/done
- See total, completed, remaining, and percent-complete progress

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
